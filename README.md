# Sistema de Reservas ScapeRoom

Este programa permite la gestión de las reservas para salas de Scape Room, facilitando la organización de las sesiones y el mantenimiento de un registro de clientes.

### Funciones principales

* **Creación de reservas**: Permite dar de alta nuevas sesiones capturando información del cliente, número de jugadores, fecha y horario específico.
* **Visualización de registros**: Ofrece la posibilidad de consultar en cualquier momento la lista completa de reservas almacenadas.
* **Eliminación de datos**: Incluye una opción para dar de baja reservas de forma sencilla seleccionándolas directamente desde el menú.
* **Registro de actividad**: El sistema guarda automáticamente un historial cronológico de las acciones realizadas, como el inicio del programa o la modificación de datos.

### Funcionamiento

El usuario interactúa con el sistema mediante un menú principal con opciones numeradas. Al seleccionar una opción, el programa guía al usuario para introducir la información necesaria o confirmar las acciones de borrado y consulta. La información se mantiene guardada para asegurar que los datos estén disponibles en futuras ejecuciones.
